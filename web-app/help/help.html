Introduction


GSword online Bible studio brings the power of JSword (http://www.crosswire.org/jsword) into web so that user can enjoy Bible study with the same tools as JSword without installing JSword. The backend engine of the GSword is JSword. And the web function was implemented using Groovy and Grails technologies.


This document explains how to use GSword.


The GSword contains the following functions:


  • One Year Bible

  • Search Bible/Advanced Search

  • Search Dictionary

  • Parallel Bibles

  • Commentary

  • Display verses

  • Daily Bread

  • Other options

  • Power Point

  • Language Switch


I created a video demonstration if it helps.


http://rock.ccim.org/gsword/biblehelp.wmv



In the following sections I will explain them one by one.


One Year Bible


The word of God is precious. You will appreciate it even more when you actually read it. How about read the whole Bible? Good idea. But it is a huge task for many folks. With a little help, you can read through the Bible in one year easily. The GSword home page displays the Chinese version Bible for the day automatically. So if you visit the site once a day and read the home page, then you will finish the whole Bible in one year. You can read other Bible translations by selecting from the drop down list.

The other way is to sign up the One year through Bible reading email program

(http://bible.ccim.org/email/bibleemaillist_zh.htm).


Bible Search


Bible Search let you search the usage of key word in the Bible. To do word search

  1. Input your key word into the box of Search Word(s):

  2. Select the Bible translation to search from the Search Bible drop down list

  3. click the Lookup button on the right of the Search Bible drop down list.


Then the Total box display the total number of matches and the content area display the first 10 matches. The next 10 or previous ten will be displayed when you click the button << or >> .


More powerful search is available when you click the Advanced Search to bring up the Advanced Search menu. On the Advanced Search page, you can do more focused search by providing for detailed search criteria and narrow down/customize search range. After you provide your search criteria, click the Search button will bring you back to the main menu with your search criteria in the Search Word(s) box. Then you can select from Search Bible drop down list your Bible translation and click the Lookup button on the right of the Search Bible drop down list.


Dictionary Search


This is how you do dictionary search:


  1. Input your key word into the box of Search Word(s):

  2. Select the dictionary from the Lookup Dictionary drop down list


Parallel Bibles


Parallel Bibles let you read/study different Bible translations in parallel. To enable this function, you just need to pick up a translation from the Parallel Bibles drop down list and then click the + button.

You can also remove the parallel Bibles by pick up a translation from the Parallel Bibles drop down list and then click the – button.


Commentaries


To use commentary, you just need to pick up a commentary from the Commentaries drop down list and click the + button on the right. You can also remove the commentaries by pick up a commentary from the Commentaries drop down list and click the - button on the right.


Display Verses


To read particular verses, you need to put the verses into the Verses box and click the Lookup button on the right. You can also choose a book from the Choose Book drop down list, then select a chapter from Choose Chapter drop down list.


Please be aware that, if the book has no chapter, the tool treat the whole book as 1 chapter. For example Jude was treated as one chapter. So when you put “Jude 1” in to Verses box and click the Lookup button, it only display the verse 1. To read the whole Jude, you need to put “Jude” in the Verses box and click Lookup.

Daily Bread

Daily Bread display the daily devotion reading material which cycle each year. Just pick the devotion book and it will be displayed.

Other Options


The tool has some other options.

  1. Strongs number. Some translations has Strongs number. When you use the translations with Strongs number, you can use the Strongs number to look up the interpretation of the word in the original Bible language-Hebrew or Greek. To see Strongs, you just click the button “Show Strongs”

  2. One verse per line. The Bible can be displayed either one verse a line or multiple verses per line. Click “Show One verse per line” button will flip between the two options.

  3. Morphology. To use Morphology, you just need to click the “Show Morphology” button.


Power Point


Power Point function generate a Power Point from the current select Bible text. You can use it for presentation right away. After click the PowerPoint button, if you do not receive PowerPoint file in a popup window in your browser. The most likely reason is your browser blocked Popup function. Please enable it.


Language Switch


The tool can display both English and Chinese interface. You can click the “Chinese” or “English” button to switch language between Chinese and English.

Browser Supporting

GSword should be able to run on any of the popular browsers like:


  • Internet Explorer

  • Firefox

  • Safari

  • chrome

  • ...


GSword not only works on browsers on Windows, Unix, Mac...etc, it even works on iPhone. So you should be able to access GSword from any where.


GSword uses Javascript/Ajax extensively. If you run into problems, please first make sure Javascript is allowed to run on your browser.


The PowerPoint function need PopUp support. If you do not receive a PopUp window for PowerPoint document download after you click the “PowerPoint” button, Please make sure PopUp is allowed on your browser.
